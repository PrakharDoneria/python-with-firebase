# Python Social Media with Firebase

This is a Python project that demonstrates the implementation of a basic social media platform using Python and Firebase as the backend. The project allows users to create an account, log in, create posts, like posts, and add comments to posts.

## Features

- User account creation: Users can sign up with their email, password, and username to create a new account.
- User authentication: Existing users can log in with their email and password.
- Post creation: Authenticated users can create new posts with text-based content.
- Like posts: Users can like posts created by themselves or other users.
- Comment on posts: Users can add comments to posts created by themselves or other users.

## Technologies Used

- Python: The core programming language used for the project.
- Firebase: Firebase is used as the backend service for user authentication and database storage.

## Prerequisites

Before running the project, make sure you have the following prerequisites installed:

- Python 3.x
- `firebase-admin` package: Install it using `pip install firebase-admin`.

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/PrakharDoneria/python-with-firebase.git
   ```

2. Set up Firebase:

   - Create a new project in the Firebase console: https://console.firebase.google.com
   - Generate a service account key for your project and download the JSON file.
   - Replace the path to the service account key file in the code (`main.py`) with the actual path.

3. Run the application:

   ```bash
   python main.py
   ```
